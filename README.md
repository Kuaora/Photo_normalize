# Photo Normalize

Простой инструмент для приведения группы фотографий к единому размеру.

## Описание

Photo Normalize — это Python-скрипт, который автоматически изменяет размер всех изображений в указанной папке до заданного разрешения (по умолчанию 400x400 пикселей). Обработанные изображения сохраняются в новую папку, исходные файлы остаются неизменными.

## Возможности

- Массовое изменение размера изображений
- Автоматическая нумерация обработанных файлов
- Сохранение результатов в отдельную папку
- Поддержка PNG формата

## Требования

- Python 3.x
- Pillow (PIL)

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Kuaora/Photo_normalize.git
cd Photo_normalize
```

2. Установите необходимые зависимости:
```bash
pip install Pillow
```

## Использование

1. Создайте папку с вашими изображениями в директории проекта
2. Откройте файл `main.py` и измените название папки в строке 8:
```python
os.chdir("Ваша_папка")  # замените "Stalin" на имя вашей папки
```

3. Запустите скрипт:
```bash
python main.py
```

4. Обработанные изображения будут сохранены в новой папке `400x400` внутри исходной папки

## Настройка

Чтобы изменить размер выходных изображений, отредактируйте строку 12 в `main.py`:
```python
resized_image = original_image.resize((ширина, высота))
```

## Примечания

- Скрипт работает с PNG изображениями
- Исходные файлы не изменяются
- Результаты нумеруются автоматически (1_400x400.png, 2_400x400.png, и т.д.)
